# Blinkit-Grocery-Sales-Outlet-Performance-Dashboard-Power-BI-
This Power BI project analyzes grocery sales data to understand outlet performance, product demand, and customer purchasing patterns. The dashboard provides insights into total sales, item category performance, outlet location impact, and sales distribution across outlet types.
Interactive visualizations enable users to analyze sales trends, compare outlet performance, and identify profitable product segments. The dashboard supports business decision-making by highlighting growth opportunities and operational improvements.
